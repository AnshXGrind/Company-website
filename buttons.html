Interactive Button Library

Beautiful hover effects and animations for modern CTAs

✨ Interactive Hover Buttons

Ripple effect with gradient shift on hover. Perfect for all CTAs.

⚠️ Deprecated: Noise Effect Button

The noise button effect has been removed in favor of consistent interactive buttons across all pages.

Size Variations

Different sizes for various contexts using data-size attribute.

Buttons with Icons/Arrows

Add arrows or icons directly in the text.

Button States

Disabled states using standard HTML disabled attribute.

Full Width Buttons

Buttons that span the full width of their container.

Usage Example

Clean, monochrome interactive buttons (no funky colors). Similar to InteractiveHoverButton from MagicUI.

<!-- Interactive Button (Default - Primary Style) -->
<button data-interactive-button>
  <span>Hover Me</span>
</button>

<!-- Glass Variant (Transparent with backdrop blur) -->
<button data-interactive-button data-variant="glass">
  <span>Learn More</span>
</button>

<!-- Outline Variant (Border only) -->
<button data-interactive-button data-variant="outline">
  <span>View Demo</span>
</button>

<!-- Large Size -->
<button data-interactive-button data-size="large">
  <span>Book a Call →</span>
</button>

<!-- Small Size -->
<button data-interactive-button data-size="small">
  <span>Quick Action</span>
</button>

<!-- With Icon/Arrow -->
<button data-interactive-button>
  <span>Get Started →</span>
</button>
<button class="btn btn-gradient">Gradient</button>
