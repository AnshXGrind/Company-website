The Challenge

The client was manually pulling data from three different carrier APis, aggregating it in Excel, and emailing PDF reports to customers. This process took their operations manager 4 hours every single day.

They needed a solution that could run automatically, scale with their growth, and provide real-time visibility to their customers without manual intervention.

The Solution

We built a custom middleware using Python and deployed it on AWS Lambda. It automatically fetches data every 15 minutes, normalizes it, and syncs it to a custom dashboard we built with React.

  • Automated Ingestion: Python scripts connecting to FedEx, DHL, and UPS APIs.
  • Real-time Dashboard: A clean interface for clients to login and track shipments.
  • Smart Alerts: AI-powered categorization of delay reasons sending automated Slack notifications.
